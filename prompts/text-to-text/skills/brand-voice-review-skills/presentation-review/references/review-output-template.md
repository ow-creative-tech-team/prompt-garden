# Review Output Template

Use this structure for PowerPoint review output.

```markdown
## Presentation Review Summary

**Title**: [Presentation title]
**Audience**: [Target audience if specified]
**Slide Count**: [Number of slides]
**Verdict**: [Approved / Approved with Minor Revisions / Needs Revision / Rejected]

### Overall Assessment
[2-3 sentences on narrative strength, brand alignment, and primary areas for improvement]

### Brand Compliance Check
**Voice and writing**: [Assessment against AP Style, explicit brand writing overrides, voice, and house style]
**Naming and references**: [Assessment of Oliver Wyman, the company, colleagues, forbidden shorthand, acronyms, citations]
**Visual identity**: [Assessment of template, logo, color, typography, charts/tables, photography/illustration; state not assessable where needed]

### Detailed Findings

#### Strengths
[Specific strengths with slide references]

#### Issues

**Slide [#] - [Critical/Major/Minor]**: [Dimension] - [Feature]
- **Element**: [Headline / Bullet point / Visual / Chart / Table]
- **Text/Visual**: "[Quoted passage or visual description]"
- **Standard**: [AP Style / Brand writing override / Oliver Wyman visual guideline / Cognitive load / Global readiness]
- **Issue**: [Why this matters]
- **Suggestion**: [Concrete revision or fix]

### Cognitive Load Assessment
[Density, pacing, slide burden, and slide-delivery fit]

### Visual Brand Assessment
[Only for visuals directly inspected. Otherwise mark relevant checks not assessable.]

### Revised Slide Examples
[For high-priority slides, provide revised on-slide text where useful.]
```
