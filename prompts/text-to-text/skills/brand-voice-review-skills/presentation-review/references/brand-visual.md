# Oliver Wyman Visual Reference

Use this only when reviewing rendered slides, PPTX theme/package metadata, screenshots, or visible slide design.

## Core Visual Signals

- Visual expression should feel clear, confident, precise, and consistent.
- PowerPoint decks should use approved 16:9 Oliver Wyman templates.

## Color

- Core colors: Midnight Blue `#000F47`, Sky Blue `#CEECFF`, White `#FFFFFF`.
- Gold 750 `#FFBF00` is an action/highlight color and should be used sparingly.
- Blue 750 `#0B4BFF` is for action/emphasis.
- Neutral 250 `#F7F3EE` is common for light backgrounds/fills.
- Do not create new colors, use gradients, transparency, shadows, effects, or too many colors.
- Check contrast for text, charts, tables, and buttons.

## Typography

- Approved serif font: headlines, section titles, large quotes/statements, large numerals, and key figures.
- Noto Sans: body copy, paragraphs, captions, labels, subheads, and supporting data points.
- Fallbacks: Noto Serif where approved serif font is unsupported; Georgia Pro Light/Georgia for headlines and Arial for body when system fonts are required.
- Do not use approved serif font for body copy, old fonts such as MMC Display, distorted type, or too many weights/sizes.

## Logo

- Use approved Oliver Wyman logo files with approved endorsement.
- Preferred placement: top-left, aligned to margin/grid.
- Maintain clear space equal to cap-height around the logo.
- Use correct logo color version:
  - Midnight Blue on light backgrounds
  - White on dark backgrounds
  - Sky Blue only in controlled Midnight Blue layouts
  - Black only for true black-and-white technical needs
- Do not stretch, distort, rotate, recolor, add effects, or alter the endorsement lockup.

## Cover Slides

- Follow standard cover style and placements.
- Use sentence case.
- Cover colors are limited to White, Sky Blue, and Midnight Blue.
- Title size: up to 66 pt without a photo; up to 54 pt with an image.
- Client logo, if present, should be similar size to but never larger than the Oliver Wyman logo, visually top-aligned with it, and right-aligned with the slide margin.
- Do not create new photo-cover layouts.

## Charts, Tables, Shapes

- Chart text should be Midnight Blue.
- Use color intentionally for emphasis/de-emphasis.
- Avoid random colors, client/non-brand colors, low contrast, and purple/green as primary chart colors.
- Use provided table styles when inspectable.
- Top row table text should be bottom-aligned; body content usually top/left aligned.
- Primary shape colors: Neutral 250, Sky Blue, Midnight Blue.
- Gold 750 and Active Blue are reserved for important highlights.
- Standard line thickness is 0.75 pt.
- Do not use the old filled arrow tip.

## Photography and Illustration

- Photography is evidence, not decoration.
- Favor authentic, focused, photojournalistic, human, expertise-led imagery with copy space.
- Avoid staged/generic people imagery, heavy filters, over-saturation, and country flag icons/images.
- Illustrations should clarify, orient, or express a specific insight.
- Use approved Explainer and Symbolic assets as provided. Do not modify, deconstruct, or recreate library illustrations.
